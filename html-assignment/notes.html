HTML Tags Overview

<h1>
It is the largest heading tag and it is used for the most important heading of the page.
<em>
It is used for slightly more important text. Screen readers will change their voice for this text.
<a>
It creates a hyperlink for the page.
<img>
It helps to inserts an image to the page.
<div>
Defines a section or division in an HTML document.
